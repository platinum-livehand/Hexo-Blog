- URL 解析，提取协议、域名和路径等信息。
- DNS 域名解析，将域名转换为 IP 地址，先查缓存，再递归查询 DNS 服务器。
- 三次握手建立 TCP 连接，如果是 HTTPS 还会进行 TLS 加密握手。
- 发送 HTTP 请求。
- 服务器处理请求。
- 服务器返回响应结果。
- 四次挥手关闭 TCP 连接。
- 浏览器解析 HTML。
- 浏览器布局渲染。
